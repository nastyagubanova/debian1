В HTTP данные передаются без шифрования, а то есть - без защиты. И злоумышленик может легко получить к ним доступ.

        Часть 2
tcpdump - простой инструмент для захвата пакетов и базового анализа. tshark - более мощный инструемнт с поддержкой фильтров и его используют для более детального разбора и точечного поиска. Когда же первый инструмент обычно используется для быстрого захвата.

        Часть 5
Base64 - это кодирование, а не шифрование и закодированные данные легко восстановить, поэтому оно не предоставляет защиту при передаче через HTTP
